## Example

::tabs

::tab antlers
```antlers
{{ session:dump }}
```
::tab blade
```blade
{{-- Using session() helper and PHP --}}
@php(dump(session()->all()))

{{-- Using Antlers Blade Components --}}
<s:session:dump />
```
