## Data Structure

The value is stored as an integer.

``` yaml
number: 42
```

## Templating

Use the variable in your templates to display the value. That's pretty much it.

```
<p>My favorite number is {{ number }}.</p>
```

```html
<p>My favorite number is 42.</p>
```
