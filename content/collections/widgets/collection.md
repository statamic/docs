## Configuring

Widgets can be added to the dashboard by modifying the `widgets` array in the `config/statamic/cp.php` file.

``` php
// config/statamic/cp.php

'widgets' => [
  'getting_started',
  [ // [tl! focus:start]
      'type' => 'collection',
      'collection' => 'blog',
      'limit' => 10,
  ], // [tl! focus:end]
],
