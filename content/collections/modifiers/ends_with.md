title: 'Ends With'
---
Returns `true` if the value ends with a given string. This comparison is case-insensitive.

```yaml
punchline: That's what she said!
```

```
{{ if (punchline | ends_with('she said!')) }}
{{ if (punchline | ends_with('your mom!')) }}
```

```html
true
false
```
