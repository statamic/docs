title: 'Remove Right'

Ensures that the string never ends with a specified string.

```yaml
urls:
  - http://statamic.com/
  - http://laravel.com/
```

```
{{ urls }}
  {{ value | remove_right('/') }}
{{ /urls}}
```

```html
http://statamic.com
http://laravel.com
```
