title: Round
---
Rounds a number to a specified precision (number of digits after the decimal point). Defaults to `0`, or whole numbers.

```yaml
pi: 3.14159265359
```

```
{{ pi | round }}
{{ pi | round(2) }}
```

```html
3
3.14
```
