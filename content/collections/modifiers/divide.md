title: Divide
---
Divide a value or another variable by your variable. Pass an integer or the name of a second variable as the parameter.

```yaml
bacon: 21
skillets: 3
```

```
{{ bacon | divide($skillets) }}
{{ skillets | divide(3) }}
```

```html
7
1
```
