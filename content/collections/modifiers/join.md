title: Join
---
Turn an array into a string by gluing together all the data with any specified delimiter. It uses a comma by default.

```yaml
tasks:
  - take a shower
  - brush hair
  - clip toenails
```

```
{{ tasks | join }}
{{ tasks | join(" + ") }} = ready
```

```html
take a shower, brush hair, clip toenails
take a shower + brush hair + clip toenails = ready
```
