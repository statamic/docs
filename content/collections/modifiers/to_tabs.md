title: 'To Tabs'

Converts all instances of a specified number of spaces in a string to tabs. `4` by default. This is a boring modifier to output examples of. Here's just a few examples on how the syntax looks.

```
{{ string | to_tabs }}
{{ string | to_tabs(4) }}
```
