title: URL Encode

URL-encodes a string. The inverse of [urldecode](/modifiers/urldecode)

```yaml
string: I just want & need $pecial characters!
```

```
{{ string | urlencode }}
```

```html
I+just+want+%26+need+%24pecial+characters%21
```
