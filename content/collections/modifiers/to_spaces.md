title: 'To Spaces'

Converts all tabs in a string to a given number of spaces, `4` by default. This is a boring modifier to output examples of. Here's just a few examples on how the syntax looks.

```
{{ string | to_spaces }}
{{ string | to_spaces(2) }}
```
