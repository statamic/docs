title: Explode
---
Breaks a string into an array of strings split on a given delimiter.

```yaml
places: Scotland, England, Switzerland, Italy
```

```
{{ places | explode(',') | ul }}
```

```html
<ul>
  <li>Scotland</li>
  <li>England</li>
  <li>Switzerland</li>
  <li>Italy</li>
</ul>
```
