title: OL

Turn an array into an HTML ordered list element.

```yaml
food:
  - sushi
  - broccoli
  - kale
```

```
{{ food | ol }}
```

```html
<ol>
  <li>sushi</li>
  <li>broccoli</li>
  <li>kale</li>
</ol>
```
