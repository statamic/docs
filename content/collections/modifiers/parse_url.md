Get information about a URL.

``` yaml
url: 'http://example.com/path?query=1'
```

``` antlers
{{ url | parse_url }}
{{ url | parse_url:host }}
```

```php
[
  'scheme'   => 'http',
  'host'     => 'example.com',
  'path'     => '/path',
  'query'    => 'query=1',
]
```

```
example.com
```
